# Audio Huong Dan Vien

Dat cac file audio thu san cho che do huong dan vien vao thu muc nay.

Ten file mac dinh:

- `him-lam.mp3`
- `doc-lap.mp3`
- `c1.mp3`
- `muong-thanh-airfield.mp3`
- `de-castries-hq.mp3`

Duong dan dang duoc map trong `js/campaign.js` qua truong `guideAudio`.

Neu ban muon doi ten file hoac dung dinh dang khac nhu `.wav`, chi can sua gia tri `guideAudio` cua tung cu diem.

Vi du:

- `guideAudio: 'assets/sounds/guides/him-lam.wav'`

Neu file chua ton tai, che do huong dan vien se bo qua audio do va tu chuyen sang dia diem tiep theo sau vai giay.
